# OBOMOCARE CBO Website

Delivering care. Restoring dignity.

OBOMOCARE is a registered Community Based Organisation serving vulnerable households in Kisii and Nyamira Counties, Kenya. This is the official website repository.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Firebase
- Recharts

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
website/
  src/              React app source
  pages/            Route pages
  components/       Shared components
  contexts/         React contexts
  services/         Firebase and external services
docs/               Documents and reference
```
